# VRDC and IDRC Data Integration Scripts for NPD
This project contains the scripts used to query VRDC and/or IDRC which are internal data warehouse systmes at CMS in order to get the data that is then later used to populate the National Provider Directory (NPD).

Pleaes see the [dev](https://github.com/DSACMS/npd_vrdc_python_projects/tree/dev) branch for ongoing development work
